Stable releases

Stable releases can be found on the Codehaus Nexus Repository Manager (which is synched with the Central Maven Repository).

Latest Snapshots

The snapshot builds of JBehave can be found on the Codehaus Nexus Repository Manager. Be sure to choose the snapshot with the latest version and the filename with the latest timestamp.

Maven Configuration

Released artifacts synched to Central Maven Repository are accessible via Maven without any additional configuration if you have direct access to the internet. If you are using a Repository Manager, such as Nexus, then consult the administrator for the Maven client configuration.
